# QuennectVPN

A VPN app for Android.

## Download

Download the latest APK from [Releases](https://github.com/startech04/QuennectVPN/releases/latest).

## About

QuennectVPN provides secure and private internet access on Android.

## Release Policy

- **Only the `latest` tag is used.** Do NOT create versioned releases (e.g., `v1.0.0`).
- When updating the APK, delete the old `app-release.apk` asset from the `latest` release and re-upload the new one. Same for `metadata.json`.
- The permanent download URL is always: `https://github.com/startech04/QuennectVPN/releases/download/latest/app-release.apk`
- Never change the asset filenames or the `latest` tag name — downstream apps depend on these stable URLs.
